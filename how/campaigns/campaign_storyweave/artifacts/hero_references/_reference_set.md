# O1 — Hero reference set: best-in-class network / graph heroes

Curated exemplar set for the P1.6 hero-visual excellence pass, per [[skill_reference_inspection]].
Goal: extract what makes a network/graph visual read as a **living, growing network you want to
belong to** (evocative + premium), not an architecture **diagram** — and what's adaptable to the
aDNA register (Tokyo-Night dark-first · SS-Ghibli warmth · cyan+purple restraint · warm/calm/honest).

## Method (honest per [[doctrine_visual_inspection]])
The strongest reference for *our* feeling already lives in-vault: the **constellation north star**

## Per-exemplar — the one thing to steal · the thing to avoid

1. **Cosmograph** — *Steal:* the graph as an honest **doorway** ("explore in the browser"), and GPU-scale that makes many nodes feel effortless. *Avoid:* the cold analytics-tool chrome + firehose density (reads "dashboard," not "belonging").
2. **Obsidian graph** — *Steal:* the emotional hook that *a graph of your own things* is beautiful and personal; nodes-as-constellation. *Avoid:* the hairball at scale (illegible; we curate to 15 connected).
3. **Nomic Atlas** — *Steal:* **depth via density + soft color-by-cluster**; a "map of meaning" you want to fly into. *Avoid:* point-cloud anonymity — our nodes are *named* vaults, not dots.
4. **Neo4j / Bloom** — *Steal:* **edges that carry meaning** (relationship-typed line styles) — the graph says something, not just "connected." *Avoid:* enterprise-tool sterility.
5. **Sigma / force-graph** — *Steal:* **hover → highlight the neighborhood** as the core interaction (ambient→doorway middle path). *Avoid:* unlabeled physics-toys that spin without meaning.
6. **ethereum.org** — *Steal:* the **warm public-good register** — "built by everyone, for the good of all" (this IS aDNA's ethos). *Avoid:* mascot-cutesy that undercuts credibility.
7. **Linear** — *Steal:* **restraint** — one accent, motion that breathes not dances, dark = premium; every pixel "on purpose." *Avoid:* gradient-for-gradient's-sake with no meaning.
8. **Stripe** — *Steal:* **gradient-mesh depth + credibility**; technical warmth. *Avoid:* over-engineered motion that taxes LCP.
9. **three.js bloom/particles** — *Steal:* the **ceiling** — additive bloom, particle fields, subtle depth/parallax that read "alive." *Avoid:* the "tech-demo" trap (flashy ≠ credible; the Movement-Skeptic gates this).
10. **north star** — *Steal:* everything — it's the operator-approved target feeling; **refine, don't reinvent** (tighten labels, contrast, mobile, add depth/interaction). *Avoid:* letting motion outrun the honest register or the perf budget.
