## Transformation Rules

### Frontmatter mapping

Vault frontmatter must be transformed to SiteForge collection schemas:

**docs collection** (concepts, patterns, comparisons, use cases):
```yaml
# Vault format                    # Site format
type: concept                  →  doc_title: "The Triad"
status: active                 →  section: "concepts"
tags: [...]                    →  order: 1
                                  description: "SEO meta description"
                                  title: "The Triad — aDNA Concepts"
```

**guides collection** (tutorials):
```yaml
# Vault format                    # Site format
type: tutorial                 →  guide_title: "Create Your First CLAUDE.md"
level: beginner                →  difficulty: "beginner"
estimated_time: "20 minutes"   →  estimated_time: "20 minutes"
                                  prerequisites: ["None"]
                                  tags: ["beginner", "getting-started"]
                                  description: "SEO meta description"
                                  title: "Create Your First CLAUDE.md — aDNA Tutorial"
```

**reference collection** (upstream docs):
```yaml
# Vault format (varies)           # Site format
                               →  ref_title: "aDNA Specification"
                                  version: "2.2"
                                  stability: "stable"
                                  description: "SEO meta description"
                                  title: "aDNA Specification — Reference"
```

### Link rewriting

- Wikilinks: `[[concept_triad]]` → `[The Triad](/learn/concepts/triad)`
- Wikilinks with alias: `[[concept_triad|the triad]]` → `[the triad](/learn/concepts/triad)`
- Relative vault paths: strip and replace with site URLs
- External links: preserve as-is

### Content adjustments

- Remove vault-only frontmatter fields (`last_edited_by`, `_migration_version`)
- Add SEO fields: `title`, `description`, `canonical`
- Strip H1 heading if it duplicates the `doc_title`/`guide_title` (Astro layouts render the title)
- Preserve all body content, callouts, code blocks, tables
