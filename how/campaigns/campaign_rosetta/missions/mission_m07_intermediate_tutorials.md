# Mission M07 — Intermediate Tutorials

## Intent

Write 3 intermediate tutorials building on M06 beginner skills. The reader now extends, customizes, and operates an aDNA vault.

## Objectives

| # | Objective | File | Status |
|---|-----------|------|--------|
| 1 | Extend the ontology — add a domain-specific entity type | `what/tutorials/tutorial_extend_the_ontology.md` | completed |

| 3 | Design a mission — decompose a task into claimable objectives | `what/tutorials/tutorial_design_a_mission.md` | completed |

## Dependencies

- M06 (completed) — beginner tutorials provide learning foundation

## After-Action Review (AAR)

- **Worked**: Intermediate tutorials smoothly escalated from beginner skills. Each tutorial builds something real: an entity type with 3 artifacts, a quality-scored context file, a mission document. The "Validate" step at the end of each tutorial provides a self-check that keeps the reader honest.
- **Didn't work**: Nothing blocked.
- **Finding**: The extend-the-ontology tutorial is the most valuable of the 9 — it teaches the skill that makes aDNA domain-specific rather than generic. It bridges the gap between "I can navigate aDNA" and "I can make aDNA work for my domain."
- **Change**: None needed.
- **Follow-up**: Advanced tutorials (M08) should culminate in federation — the skill that connects individual mastery to community participation.
