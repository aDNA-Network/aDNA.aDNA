# who/assets/backplates/ — Atmosphere backplates (reserve class)

Large ambient texture(s) for canvas/page atmosphere — the reference node holds one (`helix_grid.png`,
a faint DNA-helix grid on `#16161e`).

> **Reserve class — not yet auto-consumed.** No shipped generator or stylesheet references this dir;
> it awaits the parked interaction/motion layer (M2.5, plugin-gated) or an operator's manual canvas
> background pick. A fresh node should **skip generating these** until it has a consumer — documented
> here so the asset taxonomy round-trips honestly.

**Naming**: `<motif>.png` (lowercase snake, e.g. `helix_grid.png`).

**Generate** (when wired): wide/large format, very low-contrast motif on the dark ground — it must
recede behind node cards, never compete with them.
