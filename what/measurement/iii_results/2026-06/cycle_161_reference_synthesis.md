# Cycle 161 — reference synthesis (commons / public-good showcase lens)

Reuses the complete M5.8 exemplar set (`what/exemplars/sites/`, 10 sites inspected 2026-06-03)
through a **commons-showcase** lens — no re-fetch; the set already covers the two closest
analogs. This feeds the `/commons` hero prompt vocabulary + the `SubnetworkCard` craft.

## The exemplars that map to a public-good commons

- **[[site_huggingface]]** — closest *functional* analog (registry/marketplace + community of
  orgs/spaces). **Lift:** community made visible via *real activity + named people, not claims*;
  custom illustration over stock; each card breathes despite high overall density.
  **Avoid:** HF's *scale-as-proof* ("2M+ models," "50k+ orgs") requires HF scale — aDNA's
  4-subnetwork commons is young, so **who-not-how-many** (the Val Town pattern) is mandatory.
- **[[site_ethereum]]** — closest *conceptual* twin (movement + protocol held together).
  **Lift:** lead emotionally, validate *immediately* with credibility; role-based "get started"
  pathways; low-barrier framing ("no paperwork" → aDNA's "local-first, open, free").
- **[[site_valtown]]** — the young-scale variant: **who-not-how-many** until scale arrives.

## The cycle-161 lesson (spec-named, exemplar-confirmed)

> **Named stewards + one cited mission + one honest connect path = credible.
> Logo soup + vanity metrics = not.**

Confirmed by HF (named people > claims), Val Town (who-not-how-many), ethereum (credibility
anchor). The featured grid must *immediately back* the hero's "a commons, not a catalog" claim
with **real, cited** subnetworks, or it reads unearned (the E1 trust-score lesson).

## Rules emitted into cycle-161 craft (with provenance)

1. **Card shows the steward (a who), not a count** — HF "named people," Val Town who-not-how-many.
   → `SubnetworkCard` leads with display_name + steward + the one cited mission line.
2. **Provenance is visible, not hover-gated** — closes E4 Gap #5; renders `canonical_governance`
   (the governance anchor) as quiet factual text. *held in trust*, not a compliance badge.
3. **One honest connect path per card** — the outbound `Visit ↗` (real public_url) stands in for
   "proof," not a badge wall. The full Connect affordance lands cycle 162. (ethereum low-barrier.)
4. **Illustration over stock / no logo soup** — the bespoke hero carries the warmth; cards stay
   text-and-token, no partner logos. (HF custom-illustration; AVOID logo soup.)
5. **Hero: emotional lede + immediate credibility** — "A commons, not a catalog" (lede) backed by
   the real cited grid directly below (credibility). (ethereum sequence.)
