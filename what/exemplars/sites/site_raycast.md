# site_raycast — Raycast

> **The delight ceiling** — consumer-grade polish, motion, and micro-interaction. The reference for how far aDNA can push "exciting" while keeping clarity. The Design Critic's *Delight* dimension measures against this.

## Captured (rubric)

- **Hero:** ~15 words. "Your shortcut to everything" + "powerful productivity tools... Fast, ergonomic and reliable." Reframes value as *feeling*: "It's not about saving time. It's about feeling like you're never wasting it."
- **Sections (8):** hero+CTA (Mac/Windows) → feature pillars (Fast/Ergonomic/Native/Reliable) → extensions showcase (tabbed carousel) → AI → testimonials → automation → community → developer APIs.
- **Motion/micro-interaction (the lesson):** animated keyboard illustrations, tabbed extension carousel, glass-morphism backdrops, video thumbnails w/ hover, staggered testimonial cards. **Motion reinforces the "speed" brand promise without overwhelming.**
- **Visual:** dark-mode dominant; vibrant accents (electric blue/teal); bold-header/clean-body type contrast; high-contrast UI screenshots; 3D isometric illustrations; whitespace supports scanning.
- **Delight without clarity loss:** philosophical reframe (feeling, not metrics) + self-explanatory feature names + dense-but-not-cluttered social proof (20+ faces) + tasteful humor ("And much, much more").
- **Nav:** sticky dual-path (feature explorers + power users).

## Lift for aDNA

- **Motion as meaning, not decoration** — any aDNA motion should *say something* (e.g., a context graph composing, a node joining the network), reinforcing the thesis. Bounded by the doctrine's motion budget (entrance/scroll-reveal; `prefers-reduced-motion`; Lighthouse Perf=100).
- **Reframe value emotionally** — Raycast's "feeling" reframe models how aDNA can make "context democracy" feel *aspirational* without hype.
- **Density bands** (sparse hero → banded mid → dense showcase) map onto the doctrine's bands.

## Avoid

- Raycast's *high* motion density + glassmorphism would blow the motion budget and risk the generic-AI-aesthetic tell; aDNA takes the *principle* (meaningful motion) at a fraction of the volume.
- 8 sections is over budget; the homepage stays at 5 ±1 (Raycast is a mature consumer product with more to sell).

## Install-forward / onboarding lens (2026-07-12)

Re-inspected for the P5 **R1 gate** (headless capture + WebFetch, 2026-07-12; hero stable since June — "Your shortcut to everything"). Install-forward read: Raycast is **download-first, not command-first** — the primary CTA is **"Download for Mac" / "Download for Windows (beta)"**, with "Install via homebrew or winget" as *secondary text below* (no inline command, no copy button). Its demo-as-proof is a **static screenshot + a subtle animated keyboard overlay** — product-UI-in-motion, lightweight, never an autoplay hero video. **For aDNA (a clone-and-run workspace, no installable app) the download-card model does NOT transfer** — direct evidence for **R5**'s adaptation (no download cards; a one-line prereq note instead). The transferable signal is the **restrained motion budget**: Raycast earns "delight" with one small looping overlay, not a heavy video — a data point for keeping **R1** light (a short, muted, honest clip, not a cinematic autoplay).
