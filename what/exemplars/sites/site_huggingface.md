# site_huggingface — Hugging Face

> **The closest functional analog** to aDNA's ecosystem vision: a registry/marketplace of artifacts (models/datasets/spaces) + a community + an open ecosystem. Maps almost 1:1 to aDNA's marketplace + community + network surfaces.

## Captured (rubric)

- **Hero:** minimal marketing copy — "The AI community building the future" + CTAs ("Explore AI Apps" / "Browse 2M+ models"). The hero image *is the product* (model showcase).
- **Sections (6):** hero → "Trending this week" (Models/Spaces/Datasets tabs — **the registry itself, immediately**) → "The Home of ML" (4-col feature grid, illustrated) → "Accelerate your ML" (paid tiers) → "50,000+ organizations" (logos) → "Our Open Source" (11 library cards w/ GitHub stars) → footer.
- **Above-fold focus:** the **registry is the hero** — no marketing splash before content.
- **Visual:** neutral white/dark-toggle + accent; clean type hierarchy; **custom illustrations** per feature section (not stock); each section breathes despite high overall content.
- **Community:** woven throughout — real trending activity, "50,000+ organizations" (clickable), library stars, user profiles ("Build your portfolio"). Not a "we have a Discord" link.
- **Nav:** breadth-first mega-menu — Models/Datasets/Spaces/Docs/Enterprise/Pricing top; Community/Solutions/Website mega-dropdowns. Breadth first, depth second.

## Lift for aDNA

- **Registry-as-hero / real-content-over-fluff** — lead with the living network, not a corporate pitch. (Track B/D.)
- **Community made visible via real activity + named people**, not claims.
- **Illustration over stock**; **breadth-first mega-nav** to surface 6 tracks without overwhelming.
- **Clear free/paid separation** (relevant if aDNA ever surfaces hosted services).

## Avoid

- HF's **scale-as-proof** ("2M+ models," "50k+ orgs") *requires HF scale*. aDNA's network is young → leading with raw counts would underwhelm. Use the Val Town pattern (who-not-how-many) until scale arrives — see [[site_valtown]].
- Emoji-forward "deliberately informal" aesthetic is HF's positioning choice; aDNA's ~55/45 dial wants warmer-serious (Anthropic register), not scrappy-cute.
