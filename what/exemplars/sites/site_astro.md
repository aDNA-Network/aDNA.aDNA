# site_astro — Astro (astro.build)

> **The one-command create — and aDNA's own build stack.** A sleek open-source framework whose hero pairs a single copy-able `create` command with a calm headline. Doubly relevant: **adna.network itself is built with Astro** (the sibling forge `Astro.aDNA` governs the site's build) — the vault studies the front page of the very tool it ships on (Standing Order #8).

## Captured (rubric)

- **Hero:** ~25 words — "The web framework for content-driven websites" + "Astro powers the world's fastest marketing sites, blogs, e-commerce…". Calm, benefit-first.
- **Above-fold focus:** the **install command**, hero-adjacent, sharing the fold with the headline.
- **Install treatment (hero-level, copy button):** `npm create astro@latest` — one command, a visible copy affordance with a "Copied!" confirmation. Primary CTA **"Get Started"**; secondary "Read the docs."
- **Demo-as-proof:** a static hero illustration + a **Core Web Vitals benchmark bar chart** (Astro 66% vs WordPress 48% / Gatsby 47% / Next 30% / Nuxt 28%). **No video, no GIF** — proof is *a number + a screenshot*.
- **Social proof / registry:** "used by the largest companies," a **themes showcase** (5 featured + "browse more"), partner agencies, and OSS sponsors (Netlify, Cloudflare, Webflow).
- **Visual:** dark background / light text; clean sans-serif; accent CTAs; WebP-optimized hero; medium-dense (12 sections); entrance-only motion (View Transitions described, not autoplayed).

## Lift for aDNA

- **Single-command-create in the hero + copy affordance** — the closest twin to aDNA's shipped install pill (**R2**); a *calm* command, no swagger, matches the ~55/45 dial.
- **Openness is always *present*** (MIT + "free, open-source") even when not hero-level — supports **R3** (aDNA's hero "Open standard · MIT" eyebrow) as the *movement* variant of a signal Astro keeps in the footer.
- **Proof by benchmark + screenshot, not motion** — with Bun, the second data point that install-forward tools **don't** hero a video; reinforces keeping **R1** lightweight.

## Avoid

- The **CWV benchmark chart is a framework flex** — aDNA can't (and shouldn't) claim a comparable speed scoreboard; its proof is *agentic*, not a Lighthouse bar. Borrow the *format discipline* (one honest artifact), not the metric.
- **12 sections** exceeds the homepage 5 ±1 budget ([[front_page_doctrine]] §2).
