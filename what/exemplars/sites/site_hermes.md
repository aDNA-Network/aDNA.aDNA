# site_hermes — Hermes Agent (Nous Research)

> **The operator's tonal exemplar** and the set's north star for *progressive-revolutionary done with restraint*. A privacy-first, local-first, open-source, self-improving agent — conceptually a close cousin to aDNA's "self-governed network of agents building context for the good of humanity."

## Captured (rubric)

- **Hero:** ~40 words. "The Agent That Grows With You." Substantial whitespace around the tagline. Most messaging delivered *before* first scroll.
- **Sections (~5):** nav → hero → setup (install/config code blocks) → "See It in Action" video → features grid (6 cards, 1–2 sentences each) → footer.
- **Above-fold focus:** the headline — *the words are the focus*, no banner image competing.
- **Visual:** dark, minimal/monochromatic with accent; modern sans-serif, high contrast; single decorative hero background; features as text cards (not icons). Motion: one demo video, otherwise static.
- **Positioning:** "An autonomous agent that lives on your server, remembers what it learns, and gets more capable." Technical-yet-aspirational; infrastructure words ("server," "subagents") for technical credibility + "grows with you" humanizes.
- **Tonal balance:** ~**65 revolutionary / 35 corporate restraint** — cutting-edge language ("autonomous," "persistent memory," "self-improving") held in check by clean layout, clear hierarchy, *no hype words*.
- **Nav:** flat horizontal (logo + Docs/Portal/Desktop App); social in footer; no dropdowns. Desktop-primary.

## Lift for aDNA

- The **sparse manifesto hero**: a confident one-claim headline + breathing room is enough to carry a big idea. This is the model for the "Agentic Context Democracy" hero.
- **One demo-as-proof** ("See It in Action") instead of feature-listing — earns conviction.
- **Restraint as the carrier of revolution**: the radical idea reads credible *because* the layout is calm.

## Avoid

- Features-as-plain-text-cards can read undifferentiated — aDNA should pair each with a custom mark (per `visual-identity-v2`), not emoji.
- Desktop-primary flat nav won't scale to aDNA's 6 surfaces (docs/marketplace/community/network/labs) — we need breadth-first nav (see [[site_huggingface]]).
